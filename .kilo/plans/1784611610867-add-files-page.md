# Plan: Add Files Page — Multi-file Excel Combiner

## Goal
Implement `pages/add_files.py` so a user can upload multiple `.xlsx` files, combine them on submit, prepend a filename column (no extension), display the result, show any per-file errors, and allow downloading the combined workbook as Excel.

## Decisions
- **Sheet selection:** Read only the first/default sheet from each file.
- **Column alignment:** Union all columns across files; fill missing values with `NaN` (standard `pd.concat` behavior).
- **New column:** Insert `Source File` at index 0 with the stem of the uploaded file (filename minus `.xlsx`).
- **Error handling:** Catch errors per file during read/parse; collect into a list and display with `st.error` or `st.warning`. If any file fails, still show partial results if desired, but surface all errors before the dataframe.
- **Download format:** `.xlsx` via `openpyxl` engine using `BytesIO`.
- **UI pattern:** Use `st.form` with `st.file_uploader(multiple=True, type=["xlsx"])` and a submit button, matching existing pages (`reco2B.py`, `reco_any.py`).

## Implementation Steps

### 1. Build `pages/add_files.py`
- Imports: `streamlit as st`, `pandas as pd`, `io.BytesIO`, `datetime`, `os`.
- Page config + title.
- `st.form("add_files_form")`:
  - `st.file_uploader("Upload Excel files", type=["xlsx"], accept_multiple_files=True, key="multi_xlsx")`
  - Submit button: `st.form_submit_button("Combine Files", type="primary")`
- On submit:
  - Validate files exist.
  - Loop uploaded files:
    - Try `pd.read_excel(file, engine="openpyxl")`
    - Prepend `Source File` column: `df.insert(0, "Source File", file.name.replace(".xlsx", ""))`
    - Append to list.
    - Catch exception → add to error list with filename.
  - After loop:
    - If errors list non-empty: display errors with `st.error`.
    - If at least one df succeeded: `combined = pd.concat(all_dfs, ignore_index=True)`, display with `st.dataframe(combined)`.
    - Download button:
      ```python
      output = BytesIO()
      with pd.ExcelWriter(output, engine="openpyxl") as writer:
          combined.to_excel(writer, index=False, sheet_name="Combined")
      st.download_button(
          "Download Excel",
          data=output.getvalue(),
          file_name=f"combined_{datetime.now().strftime('%Y%m%d_%H%M%S')}.xlsx",
          mime="application/vnd.openxmlformats-officedocument.spreadsheetml.sheet",
      )
      ```

### 2. Register page in `Main.py`
- Add `"Add Files": "pages/add_files.py"` to both `all_pages` and `user_pages` dicts so the page appears in the sidebar for all user types.

## Validation
- Manual: run `streamlit run Main.py`, login, navigate to Add Files, upload 2+ xlsx files with different columns, verify:
  - Files combine correctly.
  - `Source File` column is first and matches each file's stem.
  - Errors display for a corrupt file.
  - Download produces a valid xlsx.

## Risks / Notes
- Large file sets may hit memory limits; not addressed in this plan unless requested.
- If no files uploaded on submit, show `st.warning("Please upload at least one file.")`.
